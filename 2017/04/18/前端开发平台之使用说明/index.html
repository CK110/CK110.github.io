前端开发平台之使用说明

前置技能

既然是一个前端开发平台，html，javascript/typescript，css 的基础只是还是需要的。

高级: 如果了解angular 和 rxjs 则会更好

文档说明

按照这个快速上手，过一遍。。。

界面说明

##ui/可视化的html

  1. Uibook 是集成的一个操作/组件说明文档
  2. 项目视图，Project Explorer中使用svn(对于那些懒得切换视图的人可能就是一个问题)
  3. 实际UI操作界面，这边需要说明一下怎么设置属性，怎么查看dom结构，怎么编译，怎么预览。

##ts/js
需要知道的知识:

  1. @NgModule 与 @Component
  2. @ViewChild, 选择器类似 document.getElementById(‘’)
  3. http 服务的使用，决定是否要基于angular的http封装一层来更友好的支持restFul以及jwt
  4. 路由配置(是否需要lazyload)

##css
在ui平台是使用css有3种方式:

  1. 在设计页面，基于某个组件设置样式，相当于在直接在html中写行内样式，耦合度较高
  2. 全局引用，作用于整个application
  3. angular 组件模板引用的方式，默认只作用于局部
