CSS工作记录

改了一个星期的css，已经快吐了，之前学习css 也就简单的过了一遍，工作中都是用的现成框架，都不会自己写css。这次项目中使用使用的angular2组件库primeng，实在太丑了，改了一个星期，把常用的组件样式修改成antDesign风格的，记录一下此间常用的。

常用选择器

  1. element element (中间有空格)
    div p : 选择<div>元素内部的所有<p>元素
    .classA .classB : 选择 .classA 元素内部的所有带 .classB 的元素。

  2. elementelement (中间没有空格)
    .classA.classC : 选取 带有.classA.classC 的元素，比如<div class="classA classB classC" >也能定位到。

icon绝对定位小技巧

.ui-autocomplete-token-icon{
width: 12px;
height: 12px;
position: absolute;
right: 4px;
top: 50%;
margin-top: -6px;
cursor: pointer;
}

备忘

outline:0 去除button点击后周围产生的边框效果

font-size: 12px; //字体大小
line-height: 1.5; //行间距
font-weight: normal; // 加粗

font-size: 0 !important; /splitButton去掉按钮之间的留白/
